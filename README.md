# 실행 방법

* ```npm insert``` 필요

* 입력 포함 테스트 ```node index```
* 입력 미포함 테스트 ```node test```

- [x] DELETE FROM 구현 및 출력

- [x] UPDATE 구현 및 출력

- [x] SELECT FROM 구현 및 출력
